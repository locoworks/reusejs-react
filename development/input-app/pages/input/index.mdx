## ReuseJS React Input

## Overview

Input component is the precursor of `input-group` component which provides functionality specific to input component like placeholder, disable input, autocomplete etc. Along with some additional features like **show/hide passwords** using props showPassword.
Additionally since **reusejs-input** extends **html-input** it supports all the functionality of a default browser input like types, autocomplete, form, alt, for etc.

## Installation

To use the `ReuseInput` component, install it in your project:

```bash
npm install @locoworks/reusejs-react-input
or 
yarn add @locoworks/reusejs-react-input
```

## Importing

Import the `ReuseInput` component :

```jsx
import {ReuseInput } from "@locoworks/reusejs-react-input";
```

### Form Styles

If you have been using input elements for long, especially if using tailwind for styling, you will have encountered the issue of overridding the default styles in input, especially on focus, such as ring, focus-border etc. Tailwind forms provide an easy to
use set of utilities that allow users to override these styles quite easily.

To use `form-input` tailwind class in an input element you have to follow the following steps.


1. Add `@tailwindcss/forms` to dev-dependencies.
```shell
npm install -D @tailwindcss/forms
```

2. Add to `require('@tailwindcss/forms')` to Tailwind Configuration
```json
// tailwind.config.js
module.exports = {
  theme: {
    // ...
  },
  plugins: [
    require('@tailwindcss/forms'),
    // ...
  ],
}
```

## Types/ Exported Components

The following types and exported components are available from the `@locoworks/reusejs-react-input` package:

- `ReuseInput`: The main component to create a input field.

## Props

The `ReuseInput` component is designed to function similarly to the standard HTML `<input>` tag, allowing you to create input elements within your React applications. It accepts all the props that a regular `<input>` tag does, making it a versatile and flexible component for handling user input.
